## 컬러 사용 가이드

### Primary (`#6B7C5C` 올리브 그린)
- 네비게이션 바 배경, 로고 영역
- 메인 CTA 버튼 (`구매하기`, `더 보기` 등)
- 섹션 제목 강조선, 아이콘 컬러
- 사용 원칙: 페이지당 10% 이하로 제한해 포인트 효과 유지

### Secondary (`#C8A882` 웜 베이지 브라운)
- 가격 표시, 할인율 강조
- 배지, 태그 (NEW, SALE 등)
- 인용구 또는 특별 안내 문구

### Background (`#F5F0E8` / `#EDE8DE`)
- `--color-bg-primary`: 전체 페이지 기본 배경
- `--color-bg-secondary`: 교차 섹션 배경 (짝수 섹션 등)
- 카드 내부는 반드시 `--color-bg-card` (#FFFFFF) 사용

### Text
- 제목(h1~h3): `--color-text-primary` (#2C2C2C)
- 본문 단락: `--color-text-primary` 또는 `--color-text-secondary`
- 플레이스홀더, 캡션: `--color-text-muted`
- 흰 배경 위 최소 대비비 4.5:1 준수 (WCAG AA)

### Interactive
- 버튼 기본 상태: `--color-button-primary` + 흰색 텍스트
- 버튼 호버: `--color-button-hover` (10% 어두워짐)
- 링크 밑줄은 호버 시에만 표시 권장

## 반응형 고려사항

### 다크모드 대응
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #1E1E1A;
    --color-bg-secondary: #2A2A26;
    --color-bg-card: #2E2E2A;
    --color-text-primary: #F0EBE0;
    --color-text-secondary: #B0A898;
    --color-text-muted: #706A60;
    --color-border: #3A3A36;
    --color-primary: #8A9E78;       /* 밝기 올려 가독성 확보 */
    --color-primary-dark: #6B7C5C;
  }
}
```

### 모바일 (~ 768px)
- 네비 배경색 `--color-primary` 유지, 텍스트는 흰색
- 카드 간격 축소 시 `--color-border` 구분선으로 시각 분리
- 버튼 최소 높이 44px, 터치 영역 충분히 확보

### 태블릿 (769px ~ 1024px)
- 그리드 레이아웃 전환 시 배경 교차 섹션(`--color-bg-secondary`) 활용해 가독성 유지

### 접근성
- 텍스트와 배경 대비비 최소 4.5:1 (WCAG AA) 준수
- `--color-primary` (#6B7C5C) 위 흰색 텍스트: 대비비 약 4.7:1 ✅
- `--color-text-primary` (#2C2C2C) on `#F5F0E8`: 대비비 약 11:1 ✅
